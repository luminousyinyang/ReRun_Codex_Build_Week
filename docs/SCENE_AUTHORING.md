# Scene authoring guide

## Non-negotiable rules

- Write from the learner's supplied material; do not invent contested facts or citations.
- Every incorrect answer receives a concise, pedagogically accurate correction. Never mock, shame, punish, or call the learner careless.
- Use classroom-safe language for ages 13+; no real public figures, copyrighted characters, gore, or unsafe scenarios.
- Keep each line short enough to caption and read on a television screen. Let the interaction, not exposition, carry the lesson.
- The host may be playful and Socratic but must not reveal an answer to a call-in request before the learner attempts it.

## Scene recipes

| Scene | Purpose | Required ingredients |
| --- | --- | --- |
| Recap | warm up retrieval while loading | one prior fact, accepted answers, a concept key |
| Narrative | establish a meaningful problem | visual metaphor, one factual claim, optional deep-dive/rewind levels |
| Beat | require an active response | one objective, 2-4 plausible options, one correct answer, valid destinations |
| Branch outcome | make a misconception memorable | consequence, exact refutation, a variant re-ask destination |
| Commercial | disguise spaced review as a TV trope | previously introduced concept, answer-required skip path |
| Cliffhanger | create return motivation without false urgency | next-concept teaser and a computed review time |

## Good question design

Distractors should map to realistic misconceptions, not silly choices. Prefer contrasts such as stored fuel vs. immediately usable energy, information vs. energy carrier, or by-product vs. intended product. Each variant question should assess the same objective using different wording or context.

## Rewind levels

Level one lowers jargon and changes the analogy; level two simplifies further. Both preserve the same factual claim. Rewind is not an excuse to generate novel facts, and it must retain the original scene if the live request fails.

## Author review checklist

- Can a learner answer using only the supplied material and prior scenes?
- Does the branch explain *why* the distractor is wrong?
- Is every `next`, `onCorrect`, and `onIncorrect` reference valid?
- Would the line remain supportive when read aloud to a struggling learner?
- Does the scene respect captions, reduced motion, and age positioning?

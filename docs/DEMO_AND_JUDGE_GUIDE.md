# Demo and judge guide

## Zero-key path

1. Run the app with no environment variables and power on the television.
2. Choose **The Circuit Frontier** directly from the lineup (CH 04).
3. Complete the warm-up. One wrong or empty check gives a hint; the answer appears only on the second miss.
4. Watch the first four teaching steps. Confirm the durable formula/key-term lower-third appears when authored and remains visible during the question.
5. With AUTO on, use **Skip line ▶** or the remote **▶▶ FFWD** to skim narration. Pause again to resume with the same remote control.
6. Choose a distractor. After the corrective branch finishes, choose **Rewind & retry**: it returns to the short cue and the authored simpler question, not a full re-lecture.
7. Finish the Act 2 retrieval beat, cumulative finale, and cliffhanger. The TV guide also lists saved locally generated episodes.

## Live-material path

Open **Generate an episode from your notes**. Paste text or add PDF, DOCX, PPTX, CSV, image, audio, or video. The file reader returns editable notes and a file manifest before generation. `.mov` is rejected with an export instruction; recordings over 25 MB are rejected with a trim/export instruction.

Live generation shows the rotating **Please Stand By** production statuses until a validated episode is ready. Errors return to the editor with actionable copy. Large multi-file uploads are best demonstrated locally because hosted payload limits vary.

## Accessibility checks

At 390 px wide, let a beat arrive and scroll within the question card: all answer options remain reachable. Keyboard focus is visible, captions are on by default, and reduced-motion settings suppress decorative motion.
